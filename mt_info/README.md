# mt_info

Open-source FiveM HUD for vRP servers.

## Features
- Date, weekday and time
- Server ID (ingame ID)
- vRP user_id
- Staff detection
- Job-based colors
- Toggle HUD (F10)
- Optimized & professional

## Installation
1. Place `mt_info` in your resources folder
2. Add `ensure mt_info` to server.cfg
3. Restart server
